# Recipe-App

> Recipe app | Built with MERN and GraphQL

## Getting Started

```shell
cd recipe-app
yarn
cd client
yarn
cd ..
yarn run dev
```

For running in local you need to change endpoint for GraphQL server in client/src/apollo.config.js

### Prerequisites

-   `node`
-   `npm || yarn`
